# Navigation Rapide - Deuxième Cerveau

## Fichiers Principaux
- `deuxieme_cerveau/app_new.py` - Application Flask principale
- `deuxieme_cerveau/index.html` - Interface utilisateur
- `deuxieme_cerveau/search-server-fixed.js` - Serveur de recherche Node.js
- `deuxieme_cerveau/category_path_resolver.py` - Résolution des chemins

## Scripts de Démarrage
- `deuxieme_cerveau/START.bat` - Démarrer l'application
- `deuxieme_cerveau/STOP.bat` - Arrêter l'application

## Configuration
- `deuxieme_cerveau/categories.json` - Définition des catégories
- `deuxieme_cerveau/category_mapping.json` - Mapping hiérarchique
- `deuxieme_cerveau/requirements.txt` - Dépendances Python
