========================================
  SCRIPTS DISPONIBLES
========================================

UTILISATION NORMALE:
--------------------

START.bat
  → Lance l'application complete (recommande)
  → Demarre Flask (5008) + Search (3008)
  → Ouvre le navigateur automatiquement
  → Copie depuis src/ automatiquement

STOP.bat
  → Arrete tous les serveurs
  → Libere les ports 5008 et 3008


UTILISATION AVANCEE:
--------------------

start_search_server.bat
  → Demarre uniquement le serveur de recherche
  → Port 3008
  → Utile pour debug

demarrer_recherche.bat
  → Identique a start_search_server.bat
  → Demarre uniquement le serveur de recherche


SCRIPTS OBSOLETES (NE PAS UTILISER):
-------------------------------------

start_deuxieme_cerveau.bat
  → Ancien script PowerShell complexe
  → Utilise START.bat a la place

start_deuxieme_cerveau.ps1
  → Script PowerShell avec problemes d'encodage
  → Utilise START.bat a la place


RESUME:
-------

Pour demarrer:  START.bat
Pour arreter:   STOP.bat ou Ctrl+C

C'est tout!
