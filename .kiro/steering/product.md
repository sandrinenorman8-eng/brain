## Product Overview

**Deuxi√®me Cerveau** (Second Brain) is a personal knowledge management system for capturing, organizing, and searching notes across multiple categories.

### Core Features

- **Category-based note organization** with hierarchical folder structure
- **Quick note capture** with timestamp tracking
- **Full-text search** across all notes and categories
- **File upload** support for any file type
- **Category fusion** to merge notes from multiple categories
- **Folder management** with visual interface (open, delete, merge)
- **Backup system** for complete project archival

### User Experience

The application provides a dark-themed, modern UI with:
- Real-time note saving
- Alphabetical category navigation
- File browsing with date/time sorting
- Search functionality with context excerpts
- Direct file access through system explorer

### Target Use Case

Personal note-taking and knowledge management for individuals who need to organize thoughts, ideas, projects, and research across multiple domains (business ideas, scripts, software projects, books, etc.).
