### 6.4 Validation Multi-Machines

**Checklist Validation:**
☐ Machine 1: Extension installée et configurée
☐ Machine 2: Extension installée via même méthode
☐ URL backend identique sur les deux machines
☐ Authentification réussie sur les deux
☐ Appels API fonctionnels depuis les deux
☐ Aucune erreur CORS/CSP dans consoles
☐ chrome.storage.sync: backendUrl synchronisé (si Web Store)
☐ chrome.storage.local: tokens séparés (sécurité)
