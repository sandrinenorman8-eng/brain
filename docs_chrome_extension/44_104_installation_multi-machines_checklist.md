### 10.4 Installation Multi-Machines Checklist
☐ Extension publiée sur Chrome Web Store OU
☐ Extension packagée (.crx) pour distribution manuelle
☐ Documentation installation fournie
☐ Machine 1: Extension installée et testée
☐ Machine 2: Extension installée via même méthode
☐ backendUrl identique sur machines (si sync)
☐ Authentification réussie sur chaque machine
☐ Appels API fonctionnels depuis chaque machine
☐ Tests effectués sur réseaux différents
☐ Tests effectués avec/sans VPN
☐ Aucune erreur CORS dans consoles
☐ Aucune erreur CSP dans consoles
☐ Service worker reste actif (ou restart correct)
