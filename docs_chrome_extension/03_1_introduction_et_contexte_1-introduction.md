1. INTRODUCTION ET CONTEXTE {#1-introduction}
1.1 Mission Globale
Rendre une extension Chrome capable de se connecter à un backend distant depuis plusieurs ordinateurs via une URL publique sécurisée (HTTPS), avec gestion complète de l'authentification, CORS, CSP et cas limites identifiés en 2025.
1.2 Objectifs Stratégiques

Accessibilité: Backend disponible publiquement en HTTPS
Sécurité: Authentification robuste sans secrets exposés
Portabilité: Installation facile sur machines multiples
Maintenabilité: Configuration dynamique de l'URL backend
Conformité: Respect des standards Manifest V3 et politiques de sécurité
