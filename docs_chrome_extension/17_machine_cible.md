# Glisser-déposer dans chrome://extensions
Étape 4.2.3 : Installation sur Machine Cible
1. Ouvrir chrome://extensionsRetryPContinue
Activer "Developer mode"
Glisser-déposer fichier .crx DANS la page
OU cliquer "Load unpacked" et sélectionner dossier
Accepter permissions
Vérifier icône extension dans toolbar


**Étape 4.2.4 : Configuration Post-Installation**

Clic-droit sur icône extension → Options
Saisir URL backend HTTPS
Cliquer "Enregistrer l'URL"
Tester connexion
Se connecter (OAuth)
